La Bonne Com - Stage de seconde année

Intégration web - scraping web

janvier - février 2024

L'entreprise

La bonne com est une agence de communication qui accompagne les entreprises sur la création de leurs sites web

Contexte

Durant ce stage, j'ai du travailler sur un script python permettant de récupérer le numéro de téléphone et le nom d'entreprise inscrit sur les sites web google business d'entreprises.

De plus, j'ai du construire sur Wordpress la landing page d'offre et le template des sites business.site afin de réutiliser la même structure.
