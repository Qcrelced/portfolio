Scraping Linkedin - Projet personnel

2024

Contexte

Ce projet personnel s'inspire du script python de scraping que j'ai développer lors de mon second stage chez La Bonne Com

Objectif

L'objectif est de développer un script en python permettant de faire une recherche de profil Linkedin via le prénom et le nom du personnes et de récupére son "à propos", puis de stocker ces informations dans une base de donnée.
