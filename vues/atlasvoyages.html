Atlas Voyages - Stage de première année

Création de site web vitrine

Mai - Juin 2023

L'entreprise

Atlas voyages est une agence de voyages permettant d'accompagner les voyageurs sur leurs futurs voyages, de réserver des billets d'avions, de bus, bateaux

Contexte

Durant mon ce stage, j'ai du travailler sur la refonte complète d'un site web hébergé sur wix. J'ai développé le site web en HTML/CSS et JavaScript.

L'objectif de ce site web est de permettre aux client de se renseigné sur l'entreprise et les différents services qu'elle propose comme accompagner les voyageurs, achetés des billets, renseignement...

Cette refonte doit aussi permettre de donner la possibiliter aux utilisateurs de contacter l'entreprise.
